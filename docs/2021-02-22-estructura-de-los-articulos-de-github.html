Blog Posts

Este artículo me va a servir para saber como escribir un artículo para la web luismidelgado.github.io.

Habrá una cabecera con metadatos que utilizaré para diversas opciones de los artículos. Estos datos estarán en formato Front matter escrito en yaml al inicio del archivo markdown.

Los datos serán los siguientes:

title: 'Cayman Theme — As shown in browser tab and bookmarks'
        lang:   es
        date:   December 12, 2016
        publishdate: 2021-04-08
        author:
          - Jason Long
          - Collaborator Two Name
          - Another Collaborator
        description: 'Your project description, to be shown in search engines results.'
        permalink: /2005/06/29/google-earth/
        publishdate: 2021-04-09
        image: 
        aktt_notify_twitter:
          - 'no'
        dsq_thread_id:
          - "359879364"
        categories:
         - Blog Posts
        tags:
          - your
          - keywords list
          - for
          - search engines
          - indexing
  • title: Es el título del artículo.
  • Los tags son las keywords que están enfocadas a los motores de búsqueda.
  • publishdate.

Enhorabuena por leer el artículo

Este proyecto es un aprendizaje de como crear una web estática a partir de archivos markdown y donde puedo hacer experimentos. Esto me servirá para crear grandes artículos en mi blog Kacharreando.com, donde puedes aprender a realizar proyectos por ti mismo.
