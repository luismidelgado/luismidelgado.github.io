Cómo generar un post automático con cabecera

Blog Posts

Generar artículo para el blog estático con cabecera yaml.

Este es el script en bash:

#!/bin/bash
        read -p "Enter the title: " title 
        Title=`echo $title | tr -d '[:punct:]'`
        for word in $Title
        do
          dashedTitle=${dashedTitle}-${word}
        done
        filename="`date +%Y-%m-%d`${dashedTitle}.md"
        touch $filename
        echo "---" >> $filename
        echo "published: true" >> $filename
        echo "layout: post" >> $filename
        echo "title: \"${title}\"" >> $filename
        echo "author: Yu" >> $filename
        echo "categories:" >> $filename
        echo "tags:" >> $filename
        echo "-" >> $filename
        echo "---" >> $filename
        echo "" >> $filename

Enhorabuena por leer el artículo

Este proyecto es un aprendizaje de como crear una web estática a partir de archivos markdown y donde puedo hacer experimentos. Esto me servirá para crear grandes artículos en mi blog Kacharreando.com, donde puedes aprender a realizar proyectos por ti mismo.
